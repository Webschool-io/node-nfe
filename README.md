# NFe - Node.js

Projeto ABERTO e GRATUITO para criar a NFe para Node.js 

## Sumário

- Requisitos
- Certificado Digital
- Computador
- Software
- Internet
- Campos obrigatórios e como preenchê-los:
- Dados do Emitente (você)
- Dados do Destinatário (seu cliente)
- Produto(s) ou Serviço(s) vendido(s)
- Nome do Produto ou Serviço
- Tipo
- Quantidade
- Valor Unitário
- Valor Total
- Peso Líquido Total
- Peso Bruto Total
- NCM
- CEST (Código Especificador de Substituição Tributária)
- Tributação:
- ICMS
- Situação Tributária
- Origem
- Natureza da operação
- Forma de pagamento
- CFOP (Códigos Fiscais de Operações e Prestações)
- Base de Cálculo do ICMS
- Valor Total do ICMS
- Base de Cálculo do ICMS ST
- Valor Total do ICMS ST
- Valor Total dos produtos e serviços
- Valor Total do Frete
- Valor Total do Seguro
- Valor Total do Desconto
- Outras Despesas acessórias
- Valor Total da NF-e
- Valor aproximado total de tributos federais, estaduais e municipais
